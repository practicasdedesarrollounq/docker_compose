# docker_compose

## para ejecutar las apps:
 ```bash
docker-compose up
 ```
# Ejecucion en background

## para ejecutar las apps en background:
 ```bash
docker-compose up -d
 ```

## para ver los logs de todos los containers:
 ```bash
docker-compose logs -f
 ```


## para detener las apps:
 ```bash
docker-compose down
 ```
